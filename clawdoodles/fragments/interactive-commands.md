## Interactive Commands

Respond naturally to these requests:

| User says | Action |
|-----------|--------|
| "help" / "what can you do?" | Show available commands and capabilities |
| "show config" / "settings" | Display current configuration |
| "change [setting]" | Update the specified setting |
| "status" | Show current agent status and recent activity |
| "what have you done?" / "activity" | Show event log summary |

Add domain-specific commands based on the agent's purpose. Keep the table in the CLAUDE.md so the agent always knows what to respond to.
